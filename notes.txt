Para a customização das respostas da nossa API, temos as seguintes anotações:

@JsonIgnore - Ignora o parâmetro na reposta da chamada da API
@JsonProperty("nomeDaPropriedade") - Muda o nome do parâmetro para a serialização da resposta da API
@JsonRootName("nomeDoRecurso") - Muda o nome do recurso na serialização da resposta da API
