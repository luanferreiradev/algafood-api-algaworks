Podemos criar arquivos de propriedades para diferentes profiles, basta criar um arquivo com o nome do profile, exemplo:

application-development.properties

Estes arquivos terão apenas as propriedades específicas de cada profile, o que não houver nelas será obtido do
arquivo padrão.

Para definir o profile pelo IntelliJ:
Add the VM configuration:
-Dspring.profiles.active=nomeDoProfile

Para definir o profile no arquivo application.properties:
spring.profiles.active=nomeDoProfile
